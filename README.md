## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/learnpy3.git
    cd learnpy3
    ```

2. **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

Run any script directly, for example:
```bash
python src/00_basics.py
```
or
```bash
python src/01_hello_world.py
```
